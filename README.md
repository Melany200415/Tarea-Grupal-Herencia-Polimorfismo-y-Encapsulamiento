## Descripción Breve del Ejercicio
Este proyecto contiene dos sistemas desarrollados aplicando los principios de la Programación Orientada a Objetos en Java:
1. **Sistema de Subsidio de Gasolina:** Un modelo encapsulado que evalúa y valida si un usuario aplica para un subsidio basado en ingresos, propiedades y residencia.
2. **Sistema Bancario (Reto):** Implementación orientada a objetos que hace uso de Clases Abstractas, Herencia y Polimorfismo para administrar cuentas de Ahorros, Corrientes y de Inversión, automatizando el cálculo de tasas de interés mensuales.

## Instrucciones para ejecutar el proyecto
1. Clonar este repositorio usando: `git clone https://github.com/Melany200415/Tarea-Grupal-Herencia-Polimorfismo-y-Encapsulamiento.git`
2. Abrir el proyecto en un IDE compatible con Java (IntelliJ IDEA).
3. Para probar el primer ejercicio, ejecutar la clase `Main` dentro del paquete `ec.gob.subsidio.modelo`.
4. Para probar el reto bancario, ejecutar la clase `Main` dentro del paquete `sistema.bancario`.
